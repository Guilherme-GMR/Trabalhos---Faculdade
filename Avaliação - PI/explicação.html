    • Servidor FTP

    Servidor FTP Um servidor FTP (File Transfer Protocol) é usado para transferir arquivos entre computadores pela internet ou rede local. Ele permite o envio e recebimento de arquivos de maneira eficiente, usando clientes FTP como FileZilla e Cyberduck para se conectar ao servidor.

    JavaScript

    JavaScript é uma linguagem de programação utilizada para adicionar interatividade em páginas web. Com ela, é possível criar efeitos dinâmicos, como animações, validação de formulários, manipulação de elementos HTML e muito mais.

    Docker

    Docker é uma plataforma que automatiza o desenvolvimento, a implantação e a execução de aplicações dentro de contêineres. Esses contêineres são ambientes isolados que garantem que a aplicação funcione da mesma forma em qualquer lugar, seja no computador local ou em servidores remotos.

    o Docker, é possível criar, testar e implantar aplicações de forma rápida e eficiente, tornando o desenvolvimento mais ágil e simplificando a configuração de ambientes
